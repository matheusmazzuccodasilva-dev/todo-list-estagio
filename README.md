# To-do List - Desafio Técnico de Estágio

Aplicação web simples para gerenciamento de tarefas.

## Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Editar tarefa
- Excluir tarefa
- Pesquisar tarefas
- Validação de título obrigatório
- Validação de data prevista
- Status da tarefa: Pendente ou Concluída

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- localStorage
- IntelliJ IDEA

## Como executar o projeto

```bash
npm install
npm run dev
